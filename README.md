# chess-ai

# to run game type command in terminal while in src directory:
```
python main.py
```

# controls
press 'r' to restart game \
press 'z' to undo move \
press 'w' to play as white be careful **this restarts you game** \
press 'b' to play as black be careful **this restarts you game**
# requirements
pygame - to install make sure pip is installed on system.
when pip is installed type into terminal pip install pygame
